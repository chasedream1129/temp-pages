テキストカラーを変更する

テキストカラーを任意の色に変更できます。

テキストカラーを変更する

１、色設定したいテキストオブジェクトを選択します。

２、ツールバーのテキストカラーの設定をクリックしてカラーピッカーで変更します。

テキストカラーを部分的に変更する

１、部分的に色を変更したいテキストを選択する。

今回は、「black red black」　の「red」の部分を赤くします。

２、色を変更したい部分を選択し、マウスの右ボタンをクリックします。

色を変更したいテキストをオブジェクト編集で選択し、右クリックを押して出るメニューから選択文字列の色設定を選ぶと、カラーピッカーが表示されますので、任意の色を選択します。
