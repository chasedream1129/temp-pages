選択メニュー

オブジェクトの選択に関するメニューです。

  1. すべてを選択。

    すべてのオブジェクトを選択状態にします。

  2. 選択の解除。

    オブジェクトの選択状態を解除します。

  3. 選択の反転。

    オブジェクトの選択・非選択状態を反転します。

  4. １つ前面を選択。

    描画順位で1つ前面のオブジェクトを選択状態にします。

  5. １つ背面を選択。

    描画順位で1つ背面のオブジェクトを選択状態にします。

  6. 条件を指定して選択。

    オブジェクトの条件選択ダイアログを表示します。

    条件が共通の複数オブジェクトを選択できます。
