編集メニュー

オブジェクトのコピーや貼り付けなど、編集に関するメニューです。

  1. 取り消し。

    編集による内容の変更を元に戻し、編集前の状態に戻します。

    ファイルの入出力を伴う操作（ファイルを開く/閉じる）を行うと編集の履歴は削除されます。

    それ以前の状態に戻すことはできなくなりますので注意してください。

  2. やり直し。

    元に戻した内容に対して編集をやり直し、編集後の状態にします。

  3. 切り取り。

    選択しているオブジェクトを切り取ります。

  4. コピー。

    選択しているオブジェクトをコピーします。

  5. 貼り付け。

    切り取りまたはコピーしたオブジェクトを貼り付けます。

  6. 消去。

    選択しているオブジェクトを消去します。

  7. 検索。

    オブジェクト検索ウィンドウを表示します。

  8. 環境設定。

    環境設定ダイアログを表示します。

  9. ページ設定。

    ページ設定ダイアログを表示します。
