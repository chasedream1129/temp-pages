ウィンドウメニュー

サブウィンドウの表示on/off等に関するメニューです。

  1. プロジェクト。

    プロジェクトウィンドウを表示します。

  2. オブジェクト管理。

    オブジェクト管理ウィンドウを表示します

  3. テクスチャ一覧。

    テクスチャ一覧ウィンドウを表示します。

  4. カラーセット。

    カラーセットウィンドウを表示します。

  5. ローカライズ支援。

    ローカライズ支援ウィンドウを表示します。

  6. オブジェクト検索。

    オブジェクト検索ウィンドウを表示します。

  7. オブジェクト編集。

    オブジェクトの編集ウィンドウを表示します。

  8. サブウィンドウを隠す。

    表示されているサブウィンドウを全て非表示にします。

  9. サブウィンドウをスナップする。 　　　　

    サブウィンドウをスナップします。
