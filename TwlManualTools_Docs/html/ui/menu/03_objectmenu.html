オブジェクトメニュー

新規作成やグループ化など、オブジェクトに関するメニューです。

  1. 新規ピクチャ。

    新規にピクチャオブジェクトを配置します。

  2. 新規テキストオブジェクト。

    新規にテキストオブジェクトを配置します。

  3. 新規フレーム。

    新規にフレームオブジェクトを配置します。

    フレームはフレームオブジェクトのオブジェクト編集で設定できます。

  4. グループを作成。

    選択した複数のオブジェクトをグループにします。

  5. グループを解除。

    選択したグループを解除します。

  6. 選択オブジェクトをロック。

    選択しているオブジェクトをロック状態にします。

  7. ロックを全て解除。

    ロックされている全てのオブジェクトのロック状態を解除します。

  8. 選択オブジェクトを隠す。

    選択しているオブジェクトを隠します。

    隠されたオブジェクトはプレビューで表示されなくなります。

  9. 隠したオブジェクトをすべて表示。

    隠しているオブジェクトをすべて表示します。

  10. オブジェクトを編集。

    オブジェクト編集ウィンドウを表示します。
