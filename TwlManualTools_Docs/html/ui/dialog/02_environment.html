環境設定ダイアログ

ツールの環境設定を行います。

環境設定ダイアログは、メインメニューの[編集]→[環境設定]で開けます。

  1. 描画モードを設定します。

    ※通常はDirectXモードを使用します。

  2. ビューア起動。

    プレビューの出力先を決めます。

    • IS-NITRO-EMULATORで起動。

      接続されているIS-NITRO-EMULATORでビューアを起動します。

    • IS-TWL-DEBUGGER ハードウェアで起動。

      接続されているIS-TWL-DEBUGGER ハードウェアでビューアを起動します。

    • PCで起動。

      お使いのPCでビューアを起動します。
