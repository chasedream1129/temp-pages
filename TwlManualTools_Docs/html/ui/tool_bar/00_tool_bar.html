ツールバー

ツールバーとは

ツールバーはメインウィンドウのメニューの下に表示されます。

編集中に選択されているオブジェクトの種類によって、ツールバーで編集できる内容は変化します。

ツールバーの種類

  • [ツールバー共通機能]

    各ツールバーで共通の機能。

  • [ガイド線グリッド線ツールバー]

    オブジェクトが何も選択されていないときに表示されます。

  • [テキストツールバー]

    テキストオブジェクトが選択されているときに表示されます。

  • [ピクチャツールバー]

    ピクチャオブジェクトが選択されているときに表示されます。

  • [フレームツールバー]

    フレームオブジェクトが選択されているときに表示されます。
