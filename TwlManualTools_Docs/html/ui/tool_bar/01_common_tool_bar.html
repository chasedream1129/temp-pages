ツールバー共通機能

ツールバー共通機能は、いつでも使用できる機能です。

  1. 取り消し、やり直し。

    操作の取り消し、取り消した操作のやり直し。

  2. オブジェクト境界線の表示、非表示。

    オブジェクトの範囲を示すオブジェクト境界線の表示非表示設定。

  3. オブジェクト境界線に吸着（使用不可）。
  4. ガイド線に吸着。

    オブジェクトをガイド線に吸着させる。

  5. グリッド線の表示、非表示。

    グリッド線の表示。

  6. ガイド線の表示、非表示。

    ガイド線の表示。

  7. ページ内項目の領域表示。

    ページ内項目に設定されているテキストオブジェクトの表示と、ページ内項目の大きさを表示します。

  8. ページウィンドウの表示倍率。

    ページウィンドウの表示倍率の変更。
