テキストツールバー

テキストオブジェクトが選択されているときに表示されるツールバー。

詳細設定は、テキストオブジェクト編集でも設定できます。

  1. オブジェクト位置。

    選択されたテキストオブジェクトの位置。

  2. 文字サイズ。

    テキストオブジェクトの文字サイズを設定します。 　

  3. 行そろえ。

    テキストの行をそろえます。

  4. 列そろえ。

    テキストの列をそろえます。

  5. 文字カラー。

    文字の基本カラーを設定します。

  6. 行の間隔。

    テキストオブジェクト内の文字の行間隔を設定します。

  7. 列の間隔。

    テキストオブジェクト内の文字の列間隔を設定します。

  8. テキストの禁則処理。

    禁則処理のon/offを設定します。

  9. ページ内項目設定。

    選択したテキストオブジェクトをページ内項目に設定します。
