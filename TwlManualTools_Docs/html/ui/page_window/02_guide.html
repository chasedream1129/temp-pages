目安表示

目安表示とは

ページウィンドウにはオブジェクトのレイアウトの参考にするための目安を表示できます。

目安表示には以下の４種類の表示があり、任意でon/offを選択できます。

  • グリッド線
  • ガイド線
  • オブジェクト境界線
  • ページ内項目の領域表示

グリッド線

最大拡大時に１ドット単位で区切られた線で、ページウィンドウの表示サイズによりグリッド幅が変わります。

カラーのみ設定可能で、吸着などの機能はありません。

グリッド線の設定

グリッド線の設定はページ設定ダイアログで行えます。

また、表示のon/offをツールバー共通機能で、カラー変更をガイド線グリッド線ツールバーから行えます。

ガイド線

ガイド線にはグリッド線よりより細かな設定が可能で、吸着機能があります。

設定できる内容は、カラー、縦横それぞれの開始位置と幅、吸着機能のon/offです。

ガイド線の設定

ガイド線の設定は、ページ設定ダイアログで行えます。

また、表示のon/offや吸着on/offをツールバー共通機能で、カラー変更、開始位置や線幅の設定をガイド線グリッド線ツールバーから行えます。

オブジェクト境界線

オブジェクト境界線は、それぞれのオブジェクトの大きさを示すものです。

表示のon/offのみ設定できます。

オブジェクト境界線の設定

オブジェクト境界線の設定は、ツールバー共通機能から行えます。

ページ内項目の領域表示

ページ内項目の領域表示はページ内項目で区切られ、ビューアで表示される領域を示します。

表示は全体でon/offできます。

ページ内項目の領域の表示設定

ページ内項目の領域設定のon/offは、ツールバー共通機能から行えます。
