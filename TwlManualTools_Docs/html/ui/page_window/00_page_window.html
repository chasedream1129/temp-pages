ページウィンドウ

ページウィンドウとは

オブジェクトを配置してページを作成できます。

レイアウト作業は全てこのページウィンドウで行います。

閲覧専用ページウィンドウ

プロジェクトを指定して作業している時、編集中のプロジェクト外、もしくは同じプロジェクト内であっても

編集中ではない言語のページウィンドウは閲覧専用ページウィンドウで開きます。

ページウィンドウ

  1. ページウィンドウのツールバー。

    ページタイトル、ページのサイズ、ページの背景色を設定します。

  2. オブジェクト。

    ピクチャオブジェクト、テキストオブジェクト、フレームオブジェクトなどのオブジェクトです。

    赤い枠で囲まれているオブジェクトが選択されているオブジェクトです。

    ロック状態のオブジェクトは青い枠で囲まれます。

  3. ページ表示エリア。

    ワークスペース上で、背景色（画像では白）の領域が１ページの大きさを示します。

    背景色を変更したときは、変更された色で表示されます。

  4. 状態表示。

    カーソルの位置や、選択しているオブジェクトの情報を表示します。

  5. 目安表示。

    作業用に、グリッド線、ガイド線、オブジェクト境界線、ページ内項目の領域を表示できます。

ページウィンドウでの操作方法

  • 選択操作。

    マウスの左クリックで、ウィンドウ内のオブジェクトを直接選択します。

  • 一括選択。

    マウスを左クリックしながらドラッグする事で、複数のオブジェクトを範囲指定して選択でます。

  • 追加選択。

    Shiftキーを押しながら選択操作を行うことで、オブジェクトを追加選択します。

  • 単独選択。

    Ctrlキーを押しながら選択操作を行うことで、グループ内のオブジェクトを単独で選択します。

    Shiftキーと組み合わせて追加選択することもできます。

  • オブジェクト編集ウィンドウを開く。

    オブジェクトをダブルクリックすることでオブジェクト編集ウィンドウを表示します。

  • コマンドメニューの表示。

    右クリックでオブジェクトメニューをポップアップ表示します。
