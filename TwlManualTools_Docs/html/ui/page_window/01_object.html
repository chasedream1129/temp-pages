オブジェクト

オブジェクトはページに配置する画像やテキストなどの領域を表し、ページを構成する基本単位となります。

TWL-ManualEditor では、ピクチャオブジェクト、テキストオブジェクト、フレームオブジェクト、３種類のオブジェクトを扱います。

ピクチャオブジェクト

ピクチャオブジェクトは、画像をページに配置するために使用されるオブジェクトです。

使用できる画像は、gif形式の画像ファイルです。

テキストオブジェクト

文章は、テキストオブジェクトを利用して配置します。

テキストに対して、カラー変更、サイズ変更、字間変更などのパラメータ調整を行うことが可能です。

また、テキスト内にアイコンなどの画像を埋め込む事が出来ます。

フレームオブジェクト

フレームオブジェクトは、画像を0枚、1枚、4枚のいずれかの組み合わせで、フレームを作成するオブジェクトです。
