テクスチャ一覧ウィンドウ

ページで使用している画像の一覧表示です。

画像表示形式

  1. 最新情報へ更新。

    ページで使用されている画像ファイルを最新状態にします。

  2. テクスチャ情報の表示形式を選択します。
    • 画像表示形式

      ページで使用されている画像ファイルを一覧で表示します。

    • 詳細表示形式

      ページで使用されている画像ファイルの情報をリストで表示します。

  3. ページで使用されている画像ファイル一覧。

    ページ内で使用されている画像ファイルのサムネイルを表示。

  4. 画像ファイルのパス。

    選択した画像のフルパスで、どこにファイルがあるか表示します。

詳細表示形式

  1. ファイル名。

    画像のファイル名を表示します。

  2. 幅。

    画像の幅をdot数で表示します。

  3. 高さ。

    画像の高さをdot数で表示します。

  4. 画像ファイルのイメージ表示。

    選択した画像のサムネイルの表示領域です。
