オブジェクト管理ウィンドウの操作メニュー

オブジェクト管理ウィンドウの左上の操作メニューです

マウスの右クリックでも同内容のメニューを表示できます。

  1. 切り取り。

    オブジェクトを切り取ります。

  2. コピー。

    オブジェクトをコピーします。

  3. 貼り付け。

    オブジェクトの貼り付けを行います。

  4. 削除。

    オブジェクトを削除します。

  5. グループを作成。

    複数選択されたオブジェクトをグループ化します。

  6. グループを解除。

    グループを解除します。

  7. オブジェクトをロック。

    オブジェクトをロック状態にします。

  8. オブジェクトをロック解除。

    オブジェクトのロックを解除します。

  9. オブジェクトを隠す

    オブジェクトを表示しなくします。

  10. オブジェクトを表示

    隠しているオブジェクトを表示します。

  11. プロパティ

    選択しているオブジェクトのオブジェクト編集を表示します。

  12. 選択オブジェクトを自動的に表示

    選択オブジェクトが自動的に表示されるかどうかを設定します。

    本項目が有効になっている場合、ページウインドウで、オブジェクトを選択した際、オブジェクト管理ウィンドウ上で、対応するオブジェクトが自動的に表示されます。

    ※多数のオブジェクトが配置されたページで、オブジェクト管理ウィンドウ上の表示位置を勝手に変更されたくない場合は、無効にしてください。
