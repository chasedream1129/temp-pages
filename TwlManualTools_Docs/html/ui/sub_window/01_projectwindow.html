プロジェクトウィンドウ

プロジェクトを管理するウィンドウです。

メインメニューの[ウィンドウ]→[プロジェクト]で、プロジェクトウィンドウを開きます。

プロジェクトウィンドウの見かた

  1. プロジェクトフォルダ。

    過去に開いたプロジェクトフォルダを、プルダウンで表示します。

  2. プロジェクトフォルダの指定。

    プロジェクトフォルダを指定します。

  3. プロジェクトをプレビュー。

    プロジェクト内にある、保存済みのページファイルを環境設定で指定しているプレビュー先にプレビューします。

    尚、作業中で保存されていないページは、保存するまでプロジェクトのプレビューには反映されません。

  4. 他言語データを作成する。

    他言語データ編集用に、プロジェクト内の言語フォルダを複製します。

  5. リロード。

    プロジェクト内に画像やページの追加があったのに、プロジェクトウィンドウに反映されていないときは、リロードして再読み込みするとプロジェクトウィンドウに反映されます。

  6. プロジェクト内画像ファイル一覧。

    プロジェクトのtexturesフォルダの下にある全ての画像ファイル（拡張子　.gif ）を表示します。

  7. プロジェクト内ページファイル一覧。

    プロジェクトのlayoutsフォルダの下にある全てのページファイルを表示します。

    ※ニンテンドーDSiウェア 電子取扱説明書のページの並び順は、プロジェクトウィンドウに表示されているページファイルの並び順になります。

    ※プロジェクトでのページファイルの並び順は、ウィンドウズのファイル名のソート順になりますので、並び順の変更はファイル名を変更する事で行ってください。

  8. 他言語フォルダ。

    他言語データを作成で作られた、他言語のフォルダを表示します。

  9. 他言語切り替え。

    プロジェクト内にある、他の言語フォルダに切り替えます。

ウィンドウ内のマウス右クリックメニュー

プロジェクトウィンドウ内で、マウスの右クリックを押す事で、右クリックメニューを表示します。

  1. プロジェクトにファイルを追加。

    過去に開いたプロジェクトを、プルダウンで表示します。

  2. エクスプローラーで開く。

    プロジェクトのフォルダを、ウィンドウズのエクスプローラーで開き、直接編集できます。

  3. 他言語データを閲覧モードで開く。

    プロジェクト内に他言語データがあるとき、編集しているページファイルに対応する、他言語のページファイルを開く事ができます。

    ※このメニューはlayoutsフォルダ内のページファイル上でマウスを右クリックしたときだけ表示されます。

  4. 他言語メニュー。

    プロジェクト内に他言語データがあるときは、閲覧専用ウィンドウで、編集中ではない言語のページファイルを開く事ができます。
