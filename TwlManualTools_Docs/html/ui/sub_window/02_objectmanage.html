オブジェクト管理ウィンドウ

ページ内で使用されているオブジェクトがすべて表示されており、オブジェクトを一括管理できます。

メインメニューの[ウィンドウ]→[オブジェクト管理]で、オブジェクト管理ウィンドウを開きます。

オブジェクト管理ウィンドウの見かた

  1. 操作。

    選択しているオブジェクトに対するコマンドメニューです。

    マウスの右クリックでも同内容のメニューを表示できます。

  2. 選択オブジェクトを自動的に表示on/off。

    ページウィンドウなどで選択したオブジェクトが、自動的に表示される機能をon/offできます。

    ※多数のオブジェクトが配置されたページで、オブジェクト管理ウィンドウ上の表示位置を勝手に変更されたくない場合は、offにしてください。

  3. パラメータ調整アイコン。

    オブジェクトのパラメータを調整するアイコンです。

    左側の目アイコンで、表示・非表示状態を設定します。

    ※プレビューやバイナリ出力時には、設定に関わらず、表示されます。

    右の錠前アイコンでロック状態を設定します。

  4. オブジェクトの表示。

    オブジェクトをツリー表示しています。

    表示の上に位置するオブジェクトほど手前に表示され、

    移動させる事で表示の優先順位を変更できます。

  5. リンクされた画像の状態表示。

    オブジェクトにリンクされた画像の状態を示します。

    オブジェクト内にプロジェクトの外からリンクされている画像がある。
    オブジェクト内で使用されている画像が見つからない。

オブジェクト管理ウィンドウでの操作方法

  • 選択操作。

    マウスの左クリックで、ウィンドウ内のオブジェクトを直接選択します。

  • 一括選択。

    Shiftキーを押しながら選択操作を行うことで、複数のオブジェクトを範囲指定して選択でき、

    選択したオブジェクトとオブジェクトの間が、一括で選択されます。

  • 追加選択。

    Ctrlキーを押しながら選択操作を行うことで、オブジェクトを追加選択します。

  • オブジェクト編集ウィンドウを開く。

    オブジェクトをダブルクリックすることでオブジェクト編集ウィンドウを表示します。

  • コマンドメニューの表示。

    右クリックでコマンドメニューをポップアップ表示します。

  • オブジェクト名の変更。

    グループ、オブジェクト名をクリックして選択し、RootPane 以外のグループの名前を変更できます。
