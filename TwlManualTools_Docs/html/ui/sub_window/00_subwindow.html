サブウィンドウ

メインメニューの[ウィンドウ]で開く事のできるウィンドウで、オブジェクトの状態確認や調整、その他の操作を行えます。

サブウィンドウの種類

  • [プロジェクトウィンドウ]

    プロジェクトウィンドウでは、プロジェクトの管理やプロジェクトのプレビューなどの機能が使えます。

  • [オブジェクト管理ウィンドウ]

    オブジェクト管理ウィンドウでは、ページに含まれるオブジェクトのグループの管理や、表示の優先順位の変更などの機能が使えます。

  • [テクスチャ一覧]

    テクスチャ一覧では、ページに使用されている画像ファイルの情報を見ることができます。

  • [カラーセット]

    カラーセットでは、カラー登録などの機能が使えます。

  • [ローカライズ支援]

    ローカライズ支援では、ローカライズ支援機能が使えます。

  • [オブジェクト検索]

    オブジェクト検索では、オブジェクトの検索が行えます。

  • [オブジェクト編集]

    オブジェクト編集では、オブジェクト毎の詳細設定を行えます。
