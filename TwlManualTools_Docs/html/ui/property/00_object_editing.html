オブジェクト編集

オブジェクトをダブルクリックするなどでオブジェクトの詳細を編集できる、オブジェクト編集ウィンドウを表示できます。

オブジェクト編集は、オブジェクトの種類によって、以下の３つの編集ウィンドウがあります。

  • ピクチャオブジェクト編集。

    ピクチャオブジェクトの設定を行います。

  • テキストオブジェクト編集。

    テキストオブジェクトの設定を行います。

  • フレームオブジェクト編集。

    フレームオブジェクトの設定を行います。

オブジェクト編集共通項目

  • 基本情報とユーザ拡張情報は、３種類のオブジェクト編集ウィンドウで共通の項目です。

    1. 基本情報。

      オブジェクトの名前、コメント、位置を設定できます。

    2. オブジェクト毎の設定。

      選択されたオブジェクト毎に、設定できる内容が変わります。

      オブジェクトは以下の３種類あります。

      ・ピクチャオブジェクト。

      ・テキストオブジェクト。

      ・フレームオブジェクト。

    3. ユーザー拡張情報。

      ページ内項目などの設定を行い
