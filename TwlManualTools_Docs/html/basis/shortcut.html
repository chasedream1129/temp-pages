ショートカットキー

ファイルのショートカット

キー メニュー 動作
Ctrl + N [ファイル]→[新規作成] ページを新たに作成します。
Ctrl + O [ファイル]→[開く] ページファイルを開きます。
Ctrl + S [ファイル]→[保存] ページを上書き保存します。
Ctrl + Shift + S [ファイル]→[別名で保存] ページを別名で保存します。
Ctrl + W [ファイル]→[閉じる] ページを閉じます。
Ctrl + Q [ファイル]→[終了] アプリケーションを終了します。
Alt + F4 [ファイル]→[閉じる]
[ファイル]→[終了]		 ページを閉じます。
ページが全て閉じていたらアプリケーションを終了します。

編集のショートカット

キー メニュー 動作
Ctrl + Z [編集]→[取り消し] 編集前の状態に内容を戻します。
Ctrl + Shift + Z [編集]→[やり直し] 元に戻した編集をやり直します。
Ctrl + X [編集]→[切り取り] 選択しているオブジェクトを切り取ります。
Ctrl + C [編集]→[コピー] 選択しているオブジェクトをコピーします。
Ctrl + V [編集]→[貼り付け] 切り取りまたはコピーしたオブジェクトを貼り付けます。
Del, BackSpace [編集]→[消去] 選択しているオブジェクトを消去します。
Ctrl + F [編集]→[検索] 検索ウィンドウの表示

オブジェクトのショートカット

キー メニュー 動作
Ctrl + G [オブジェクト]→[グループを作成] 選択した複数のオブジェクトをグループにします。
Ctrl + Shift + G [オブジェクト]→[グループを解除] 選択したグループを解除します。
Ctrl + 2 [オブジェクト]→[選択オブジェクトをロック] 選択したオブジェクトをロック状態にします。
Alt + Ctrl + 2 [オブジェクト]→[ロックをすべて解除] 全てのオブジェクトのロック状態を解除します。
Ctrl + 3 [オブジェクト]→[選択オブジェクトを隠す] 選択したオブジェクトを隠します。
Alt + Ctrl + 3 [オブジェクト]→[隠したオブジェクトをすべて表示] 隠していたオブジェクトをすべて表示します。
F12 [オブジェクト]→[オブジェクトを編集] オブジェクト編集ウィンドウを開きます。

選択のショートカット

キー メニュー 動作
Ctrl + A [選択]→[すべてを選択] 全てのオブジェクトを選択状態にします。
Ctrl + Shift + A [選択]→[選択の解除] オブジェクトの選択状態を解除します。
Ctrl + Alt + ] [選択]→[１つ前面を選択] オブジェクトの一つ前面のオブジェクトを選択します。
Ctrl + Alt + [ [選択]→[１つ背面を選択] オブジェクトの一つ背面のオブジェクトを選択します。

表示のショートカット

キー メニュー 動作
Ctrl + Y [表示]→[アウトライン表示] オブジェクトの境界線だけ表示します。
Ctrl + U [表示]→[オブジェクト境界線に吸着] オブジェクトの境界線に吸着します（未実装）
Ctrl + Shift + + H [表示]→[オブジェクト境界線を隠す] オブジェクト境界線を隠します。
Ctrl + H [表示]→[グリッドを表示] グリッドを表示します。
Ctrl + R [表示]→[ガイドに吸着] ガイド線に吸着します。
Ctrl + E [表示]→[ガイドを表示] ガイド線を表示します。
Ctrl + + [表示]→[ズームイン] 表示倍率をアップします。
Ctrl + - [表示]→[ズームアウト] 表示倍率をダウンします。
Ctrl + Shift + + [表示]→[ウィンドウ拡大] ページウィンドウを拡大します。
Ctrl + Shift + - [表示]→[ウィンドウ縮小] ページウィンドウを縮小します。
A [表示]→[全体表示] 表示倍率を 100 ％に戻し、ページのレイアウト全体が表示されるようにページウィンドウの大きさを変更します。
F [表示]→[選択部分を全体表示] 選択したオブジェクトがウィンドウ全体に表示されるように表示倍率を変更します。

ウィンドウのショートカット

キー メニュー 動作
Ctrl + P [ウィンドウ]→[プロジェクト] プロジェクトウィンドウを表示します。
F10 [ウィンドウ]→[オブジェクト管理] オブジェクト管理ウィンドウを表示します。
F8 [ウィンドウ]→[テクスチャ一覧] テクスチャ一覧ウィンドウを表示します。
Alt + F7 [ウィンドウ]→[カラーセット] カラーセットウィンドウを表示します。
Ctrl + L [ウィンドウ]→[ローカライズ支援] ローカライズ支援ウィンドウを表示します。
Ctrl + F [ウィンドウ]→[オブジェクト検索] オブジェクト検索ウィンドウを表示します。
F12 [ウィンドウ]→[オブジェクト編集] オブジェクト編集ウィンドウを表示します。
F6 [ウィンドウ]→[サブウィンドウを隠す] 表示されているサブウィンドウの表示・非表示を切り替えます。

プレビューのショートカット

キー メニュー 動作
F5 [プレビュー]→[ページを保存せずにプレビュー] アクティブなページウィンドウを、保存せずにビューアでプレビューします。
Shift + F5 [プレビュー]→[プロジェクトをプレビュー] 保存されているプロジェクトをビューアでプレビューします。

ヘルプのショートカット

キー メニュー 動作
F1 [ヘルプ]→[ヘルプ] ヘルプを表示します。
